# Config
See `config.schema.yaml` for descriptions.

## Example configuration
* `config_primates.yaml`
    * Configuration for primate samples.
    * This example was tested on the [T2T primates](https://github.com/marbl/Primates).
    * __CenMAP will only work on non-human primate species with alpha-satellite repeats.__
* `config_tumor.yaml`
    * Configuration for human tumor samples.
    * This example was used for [HG008](https://www.nist.gov/programs-projects/cancer-genome-bottle).
* `config_minimal.yaml`
    * Configuration for minimal workflow. Only detects putative active HOR arrays.
